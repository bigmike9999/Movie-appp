# Movie App (Vite + React)

Simple movie/TV show listing with the ability to add and filter movies by title and rating.

Quick start (PowerShell):

```powershell
cd "c:\Users\michael alabi\Desktop\Movie-app"
npm install
npm run dev
```

Files of interest:
- `src/App.jsx` - main app, add form and filtering
- `src/components/MovieCard.jsx` - single movie card
- `src/components/MovieList.jsx` - list of movies
- `src/components/Filter.jsx` - filter controls (title, rating)
